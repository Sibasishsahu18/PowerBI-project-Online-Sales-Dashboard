Power BI – Online Sales Dashboard
• Project Summary: Developed an interactive dashboard to analyze online sales across regions,
products, and time periods using Power BI.
• Key Responsibilities & Achievements: Performed data cleaning and modeling using Power Query &
DAX, created dynamic KPIs, and implemented drill-down visualizations with slicers and filters.
• Impact: Enabled deeper sales insights, helping identify revenue trends and top-performing
segments for better business decisions.
